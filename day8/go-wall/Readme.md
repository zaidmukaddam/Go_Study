## Go Unsplash Wallpaper   
Set Wallpaper from Unsplash API     

### Create env file
`cp .env.example .env`

### Write env file
Create an application on [
Unsplash Developers](https://unsplash.com/developers)   

And get the **Access Key**
```
UNSPLASH_ACCESS_KEY=<Access Key>

```

### Run app
go run .
